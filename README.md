## Testing and Running

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Second, run the Json Server to get the Episodes List like an API:

```bash
npm run server
# or
yarn server
```

<hr />

## To test GetStaticPaths

GetStaticPaths just work in Production Mode (only after build):

```bash
yarn build
# then
yarn start
```
